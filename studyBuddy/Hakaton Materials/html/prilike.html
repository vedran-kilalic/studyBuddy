Otvorene prijave za internship u kompaniji Code Red

Obrazovanje i društvena odgovornost sastavni su dio Code Red kulture. Kompanija je posvećena pružanju podrške budućim IT profesionalcima u izgradnji njihovih karijera, te već devetu godinu provodi programe nase prakse koja se održava do tri puta godišnje. Ova praksa tromjesečni je program edukacije namijenjen svim osobama koje svoje teorijsko znanje žele primijeniti u praksi. Praktikanti imaju priliku raditi na projektu uz individualno mentorstvo iskusnih Code Red inženjera te, pored sticanja praktičnog iskustva, svi praktikanti zadržavaju vlasništvo nad svojim radom.
